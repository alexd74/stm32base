General

This page describes MSP430 scmRTOS release for msp-gcc toolchain.

Release consists of the RTOS sources and three sample projects - see Release description for details. Each sample located in separate folder. Structure of sample folder (folders are in brackets '[ ]'): 

 [cfg]             - project config files (if needed)
 [doc]             - project docs (if any)
 [exe]             - executable product[s]
 [lst]             - listings and map files
 [obj]             - objects and other intermediate files
 [src]             - project sources
 makefile          - makefile for GNU make

Structure of the RTOS folder: 

 scmRTOS
   Common            - common RTOS sources
   MSP430            - portable part

See Distribution section and below about release using manners.

Building

To build sample project just type make on command prompt. Make sure that make utility and msp-gcc's bin folder are inÂ %PATH% environment variable.

When using some IDE (like Eclipse or Code::Blocks) - just set up IDE to use this makefile. Or, if you need to specify commandlines for compiler, assembler and linker, use this settings (1):

Assembler 

 -mmcu=msp430x149
 -O2
 -g
 -I../scmRTOS/Common
 -I../scmRTOS/MSP430
 -D_GNU_ASSEMBLER_
 -DL_reset_vector__ -DL_ctors
 -Wall

Compiler 

 -mmcu=msp430x149
 -O2
 -g
 -I../scmRTOS/Common
 -I../scmRTOS/MSP430
 -funsigned-char 
 -fno-exceptions 
 -fno-rtti
 -Wall
 -Wno-reorder

Linker: 

 -mmcu=msp430x149

[1] this is for MSP43F149 chip.
