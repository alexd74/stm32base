STM32 IAR Release

From scmRTOS

General

This page describes STM32 scmRTOS release for IAR EWARM toolchain.

See Distribution section and below to learn about release using manners.

Release consists of the RTOS sources and four sample projects (see Release description for details). Each sample located in separate folder. Release structure is follows (folders are in brackets '[ ]'): 

 [1-EventFlag]    - sample
 [2-Message]      - sample
 [3-Channel]      - sample
 [4-Debug]        - sample
 [scmRTOS]        - RTOS folder

Structure of the RTOS folder: 

[scmRTOS]
  [Common]           - common RTOS sources
  [CortexM3]         - portable part
  [Extensions]       - RTOS extensions

Structure of the sample folder: 

 [Debug/Exe]       - executable product[s]
 [Debug/List]      - listings and other temporary text files
 [Debug/Obj]       - objects and other intermediate files
 [settings]        - IAR IDE auxiliary files
 [Src]             - test project sources
 slon.ewd          - IAR IDE auxiliary file
 slon.ewp          - IAR IDE project
 slon.eww          - IAR IDE workspace

slon is test project name.

Building

There is only one way to build any sample provided - IAR IDE.

Just launch IDE, open appropriate workspace (*.eww file) and execute RebuildAll command.
