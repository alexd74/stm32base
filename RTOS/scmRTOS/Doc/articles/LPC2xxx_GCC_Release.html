General

This page describes LPC2xxx scmRTOS release for GCC toolchain. Tested on Sourcery G++ Lite 2010.09-51 (GCC 4.5.1) release. Other ARM GCC toolchains should work too, possible with some linker script corrections.

Release consists of the RTOS sources and three sample projects - see Release description for details. Each sample located in separate folder. Structure of sample folder (folders are in brackets '[ ]') is follows: 

 [exe]             - executable product[s]
 [lst]             - listings and map files
 [obj]             - objects and other intermediate files
 [prj]             - linker scripts and other project config files
 [src]             - project sources
 makefile          - makefile for GNU make

If folders exe, lst and obj are not present, they are created during build process.

Structure of the RTOS folder: 

 scmRTOS
   Common            - common RTOS sources
   ARM7              - portable part
   Extensions        - OS extensions

See Distribution section and below about release using manners.

Building

To build sample project just type make on command prompt. Make sure that make utility and arm-gcc toolchain's bin folder are in PATH.

When using some IDE (like Eclipse or Code::Blocks) - just set up IDE to use this makefile. Or, if you need to specify commandlines for compiler, assembler and linker, use this settings (1):

Assembler 

 -mcpu=arm7tdmi
 -mthumb-interwork
 -O2
 -g
 -I../scmRTOS/Common
 -I../scmRTOS/ARM7
 -I../scmRTOS/Extensions/Profiler

Compiler - c files 

 -mcpu=arm7tdmi
 -mthumb-interwork
 -O2
 -g
 -std=gnu99
 -ffunction-sections -fdata-sections
 -Wall -Wextra
 -Wimplicit -Wpointer-arith -Wredundant-decls
 -Wshadow -Wcast-qual -Wcast-align -Wnested-externs -pedantic
 -I../scmRTOS/Common
 -I../scmRTOS/ARM7
 -I../scmRTOS/Extensions/Profiler

Compiler - cpp files 

 -mcpu=arm7tdmi
 -mthumb-interwork
 -O2
 -g
 -fno-exceptions -fno-rtti
 -ffunction-sections -fdata-sections
 -funsigned-bitfields -fshort-enums
 -Wall -Wextra
 -Winline
 -Wimplicit
 -Wpointer-arith -Wredundant-decls
 -Wshadow -Wcast-qual -Wcast-align -pedantic
 -I../scmRTOS/Common
 -I../scmRTOS/ARM7
 -I../scmRTOS/Extensions/Profiler

Linker 

 -mcpu=arm7tdmi
 -nostartfiles
 -Wl,--gc-sections
 -T./prj/LPC2148-ROM.ld

[1] this is for LPC2148 chip.
